# Sharepoint Cambouis

Ce projet utilise l'architecture **BMAD Fast** pour accélérer le développement assisté par IA.

## Architecture & Outils

* **BMAD Fast** est installé dans ce dépôt.
* Les agents interactifs (Fast, Auto, Secu) sont accessibles via les outils/commandes locales.
* Les résultats d'exécution et les artefacts de planification sont stockés dans le dossier `_bmad-output/`.
